# Flight Booking System

This is a flight booking system simulation app built with Flask and SQLite.

## Features

- Search for flights by departure city, destination city, and departure date.
- View a list of available flights with details.
- Book flights and receive a booking confirmation.
- View your booking history.
- Cancel bookings.

## Setup

1. **Clone the repository:**
   ```
   git clone <repository-url>
   ```
2. **Install the dependencies:**
   ```
   pip install -r requirements.txt
   ```
3. **Seed the database:**
   ```
   python seed.py
   ```
4. **Run the application:**
   ```
   python app.py
   ```
5. **Open the application in your browser:**
   [http://127.0.0.1:5000](http://127.0.0.1:5000)
